이 ZIP은 현재 대화에서 누적 요청된 기능 중 서로 충돌이 적고 비교적 안전하게 합칠 수 있는 파일만 모은 누적 패치입니다.

포함 핵심:
- COMPANY_ADMIN 1명 제한
- 외부/협력사 계정 다중 현장 지정
- 계정 삭제 보강
- 전자결재 승인함 /approvals
- 모바일 메뉴 열림/닫힘 보강
- 회사명 상단 표시
- 생산관리 대시보드 + 발주물량/발주일/출고일/공정률 계산 헬퍼

주의:
- 현장상세에 직접 꽂는 고위험 변경은 이번 누적본에서 제외했습니다. 이전 패치들과 충돌 가능성이 높아서 최신 실소스 없이 합치면 Render가 다시 깨질 수 있기 때문입니다.
- 덮어쓰기 후 Render 전에 로컬/브랜치에서 확인 권장.
